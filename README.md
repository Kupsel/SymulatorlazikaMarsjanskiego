# Symulator Łazika Marsjańskiego

## Opis projektu
Jest to symulator wyprawy łazika marsjańskiego w dwuwymiarowym świecie.
Gracz steruje ruchem łazika, który porusza się po planszy, zużywa energię i napotyka losowe zdarzenia oraz elementy świata.
Celem symulacji jest dotarcie do wyznaczonego punktu przy zachowaniu energii i w ograniczonej liczbie kroków.

## Funkcje programu
- ruch w 4 kierunkach (N, S, E, W)
- system energii
- granice świata
- losowe zdarzenia
- elementy świata (np. stacja ładowania, skały, wrak statku)
- system punktów
- raport końcowy
- możliwość ponownej gry
